题解 | Leetcode 76 - Minimum Window Substring 最小覆盖子串

March 14, 2019

审题

给定一个字符串 S 和一个字符串 T，请在 S 中找出包含 T 所有字母的最小子串。

示例：

输入: S = "ADOBECODEBANC", T = "ABC"
输出: "BANC"

说明：

如果 S 中不存这样的子串，则返回空字符串 ““。 如果 S 中存在这样的子串，我们保证它是唯一的答案。

思路

Tips

题目下方给出了两个提示：

提示一：

Use two pointers to create a window of letters in S, which would have all the characters from T.

提示二：

Since you have to find the minimum window in S which has all the characters from T, 
you need to expand and contract the window using the two pointers and 
keep checking the window for all the characters. This approach is also called Sliding Window Approach. 

L ------------------------ R , Suppose this is the window that contains all characters of T 
                          
        L----------------- R , 
this is the contracted window. We found a smaller window that still contains all the characters in T

When the window is no longer valid, start expanding again using the right pointer.

算法已经表述清楚了，解决该题的核心是使用两个指针，维护一个动态的“滑动窗口”，从前往后搜索，使得处于该滑动窗口的字符串包含所有 T 中的字符串， 找到符合的子串后，尝试将左边的指针往右移动，在满足题设条件的情况下，使得滑动窗口尽可能小

为了方便描述，我们定义以下数据结构：

  • sLen： 字符串 S 的长度
  • tLen： 字符串 T 的长度
  • slow：左指针，滑动窗口的左边界
  • fast：右指针，滑动窗口的右边界
  • tMap：数组哈希表，保存 T 中字符出现的频次
  • window： 数组哈希表，保存滑动窗口里各个字符的频次
  • letterCnt： 整数，记录滑动窗口内包含的、满足条件的字符的个数，当 letterCnt 等于 T 的长度时，说明我们至少已经找到了一个包含 T 中所有字符的解
  • minLen：整数，滑动窗口的长度
  • res：字符串，算法返回值

1.维护两个哈希表，哈希表 tMap 记录 T 中每个字符出现的次数， 哈希表 window 记录滑块间每个 map 字符出现的次数;

2.遍历字符串 S ，使用两个指针 slow, fast 数组下标记录滑块的左右边界，该滑块滑动时遵循规则如下：

3.使用 letterCnt 记录 window 找到的 T 中的字符出现的次数，当 letterCnt >= len(T) 时，说明已经找到了符合条件的子串；(注意坑：字符数量也是必要条件)

4.但该子串可能不是最小的，因此找到子串后，需要尝试将slow右移：如 slow 指向的是目标字符，且在 window 表的值大于 map 中对应的字符的频次时，slow 右移；如非目标字符，继续右移，因为这并不会改变window中满足字符的频次；

5.更新 res 和 min Len。

6.fast右移，寻找更短的滑块。重复 3 ～ 5，直到 fast 到达 S 末端。

伪代码

sLen <- len(S)
tLen <- len(T)
tMap, window := [128]int{}
for _, c :=  range T {
	tMap[c]++
}
slow, fast <- 0
// 遍历字符串 S
for fast, c :=  range S {
	if c in T {
		if count(c) in window < count(c) in tMap {
			letterCnt++
		}
		window[c] ++ 
	}
	if letterCnt == tLen {
		// try to shorten the window
		
		ch <- S[slow]
		for ch not in T or count (ch) in window > count(ch) in tMap {
			if ch in tMap {
				window[ch]--
			}
			slow++
			ch <- S[slow]
		}
		len <- 	fast - slow
		if minLen > len {
			update minLen and res
		}
	} 
}

实现

func minWindow(s string, t string) string {
	if s == "" || t == "" {
		return ""
	}
	tMap := [128]int{0}
	window := [128]int{0}
	letterCnt := 0
	slow := 0
	tlen := len(t)
	minLen := len(s) + 1
	res := ""
	for _, c := range t {
		tMap[c]++
	}
	for fast, c := range s {
		if tMap[c] != 0 { // 遇到一个目标字符，
			if window[c] < tMap[c] { // 未收集够
				letterCnt++
			}
			window[c]++
		}
		if letterCnt == tlen { // window 已经收集够了目标字符
			ch := rune(s[slow])
			for tMap[ch] == 0 || tMap[ch] < window[ch] {
				if window[ch] != 0 {
					window[ch]-- // 剪掉目标字符，因此该字符频次减1
				}
				slow++
				ch = rune(s[slow])
			} // 不合法，结束本轮搜索
			l := fast - slow
			if minLen > l {
				minLen = l
				res = s[slow : slow+l+1]
			}
		}
	}
	return res
}
