func main() {
	var wg sync.WaitGroup
	wg.Add(2)
	go func() {
		time.Sleep(2*time.Second)
		fmt.Println("1号完成")
		wg.Done()
	}()
	go func() {
		time.Sleep(2*time.Second)
		fmt.Println("2号完成")
		wg.Done()
	}()
	wg.Wait()
	fmt.Println("好了，大家都干完了，放工")
}

使用 sync.WaitGroup 能够优雅地实现等待多个 goroutine 完成。

CentOS离线安装MySQL

数据库 运维

如果你的 CentOS 运行在内网环境访问不了公网，内网环境由没有配置相关的源，那么可以考虑离线安装。

查看 CentOS 版本

1

# cat /etc/centos-release

下载rpm

repo.mysql.com/yum

选择对应的版本下载：

  • libs
  • common
  • client
  • server

安装

1

# yum localinstall mysql-xxxx.rpm

注意需要client、server依赖common、libs，因此四个包都需要安装。

TRICK 1 LIMIT X OFFSET Y

limit 是限制 SQL 执行的结果，一般是后面紧接一个正整数作为参数，但还可以添加一个 OFFSET (游标),例如：

1

limit 4 offset 3 --意味从行3取4行

TRICK 2 使用限制表名

在SQL语句中，如果有必要，可以指定执行SQL的数据库和表：例如，操作 a 数据库中的 b 表的 c 列

1

select b.c from a.b
